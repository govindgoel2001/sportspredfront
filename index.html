Win probability is a logistic function of the ELO gap. Watch the marker ride the curve as the opponent's rating sweeps — the steep middle is where small rating gaps swing the most probability.

03
Dixon-Coles xG Model
src/models/dixon_coles.py
Each team's xG drives a Poisson goal distribution. Their outer product builds a full scoreline matrix — every cell recomputes each frame as the λ values breathe. Cells sum into Home / Draw / Away.
Poisson goals → scoreline matrix live

04
ML Ensemble
src/models/ml_ensemble.py
Three calibrated models each emit a probability that wanders with inference noise. A soft-vote with sport-tuned weights (25 / 37.5 / 37.5) collapses them into one stable ensemble output.
RF + XGBoost + LightGBM → soft-vote live

Statistical Blend
src/models/calibrator.py
ELO, Dixon-Coles and H2H merge into a base estimate; the ML ensemble overlays 70/30; news sentiment applies a final ±10% nudge. Every weighted contribution flows visibly into the result.

9-Agent AI Debate

Nine specialist Claude agents each plant a probability estimate on the axis. Marker size = agent weight, glow = confidence. The consensus marker is their confidence-weighted mean, sliding as they debate.

Edge Calculation + Kelly Sizing
src/market/edge.py · kelly.py
Model probability minus the devigged market probability is the edge. As the gap moves it crosses the STRONG / MODERATE / WEAK / NEGATIVE thresholds live, and quarter-Kelly resizes the stake.
edge = P(model) − P(market) → Kelly stake live
